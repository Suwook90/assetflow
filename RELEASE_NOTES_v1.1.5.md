# FilePilot AI 1.1.5 Release Notes

## 핵심 수정
- 문서 번역/정리용 로컬 AI Pack을 다국어 문서 변환에 더 적합한 모델로 변경했습니다.
- `영문`, `영어`, `한국어` 같은 지시어만 AI가 반환하는 경우 파일 저장을 차단하고 명확한 안내를 표시합니다.
- 선택 문서 번역 요청의 대기 시간을 늘려 첫 응답이 느린 환경에서도 실패를 줄였습니다.
- 문서 에이전트 기준에 번역 모드를 추가했습니다.

## 사용 참고
- 기존 버전에서 AI Pack을 사용하던 PC도 1.1.5에서 첫 문서 번역 요청 시 새 로컬 AI 파일을 다시 다운로드합니다.
- 문서 번역 AI Pack은 첫 다운로드 시 약 1GB 내외의 저장 공간과 시간이 필요할 수 있습니다.
- TXT, JSON, MD, CSV, XML, YAML, LOG, INI, PY, JS, TS, HTML, CSS 같은 텍스트 문서만 앱 안에서 바로 번역 저장합니다.
