파일정리함 (AssetFlow) 1.0.0

첫 정식 공개 릴리스입니다.

파일정리함은 로컬 PC의 폴더, 파일, 이미지, 문서를 빠르게 확인하고 정리하기 위한 Windows용 데스크톱 도구입니다. 공개 저장소는 제품 소개와 설치 파일 배포용이며, 소스코드는 비공개 저장소에서 관리합니다.

## 주요 기능

- 파일 뷰어/이동 모드
- 이름/확장자 일괄 변경 모드
- 이미지 용량 압축 모드
- 이미지 해상도 변경 모드
- 문서 편집/일괄 변경 모드
- 모드 선택 첫 화면과 상단 모드 전환
- 기준 폴더 선택, 폴더 트리, 검색 결과 그리드, 큰 미리보기 화면
- 왼쪽/오른쪽 패널 접기, 펼치기, 드래그 크기 조정
- 대상 폴더 목록 저장, 선택 이동, 전체 이동, 대상 폴더 비우기
- 다중 선택, Ctrl/Shift 선택, 드래그 박스 선택
- `Del` 삭제, `F2` 이름 바꾸기, `Ctrl+S` 문서 저장
- 우클릭 메뉴: 열기, 위치 열기, 이미지 뷰어로 보기, 프로그램으로 보내기, 새 폴더, 복사, 잘라내기, 붙여넣기, 이동, 삭제, 경로 복사, 속성
- JPG, JPEG, PNG, WebP, BMP, GIF, TIFF, AVIF, HDR, HDRI, EXR 이미지 확인
- TXT, JSON, MD, CSV, XML, YAML 문서 확인 및 편집
- EXR depth map 흑백 미리보기와 HDR/HDRI 미리보기
- 검색 형식 프리셋과 커스텀 확장자 설정
- 이름순, 용량순, 날짜순, 경로순 정렬
- 큰 폴더용 300개 우선 로드, 백그라운드 순차 로드, 보이는 썸네일 우선 로딩
- 빠른 화살표 이동 시 이미지 미리보기 안정화
- 이미지 크기/아이콘 크기 Ctrl+휠 조정
- 문서 개별 편집/저장, 단어 일괄 변경/삭제, `.bak` 백업
- 변경 기능 목록 드래그앤드롭 추가, 선택 비우기, 실제 파일 삭제 구분
- Windows 설치 프로그램, 시작 메뉴/바탕화면 바로가기, 작업표시줄 고정 안내
- 사용자 표시명 `파일정리함`, 배포 파일명 `AssetFlowSetup.exe`

## 저장소 구성

- `Suwook90/assetflow`: 공개 다운로드/소개용 저장소입니다. 소스코드는 포함하지 않습니다.
- `Suwook90/assetflow-private`: 비공개 소스/빌드 작업용 저장소입니다.
