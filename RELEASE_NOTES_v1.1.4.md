# FilePilot AI 1.1.4

문서 변경 모드에서 선택 파일 기준과 번역 요청 인식을 개선한 안정화 배포입니다.

## 주요 변경

- 문서 모드에서 `해당파일 영문으로 바꿔줘`, `영어로 번역해줘`, `한국어로 바꿔줘` 같은 표현을 AI 문서 변환으로 인식합니다.
- 새 문서를 선택하면 이전 대화에서 고정된 문서보다 현재 선택한 문서를 우선합니다.
- `xlsx` 같은 서식 문서를 선택한 상태에서 번역을 요청하면 `내용이 비어 있습니다`가 아니라 지원 범위를 명확히 안내합니다.
- 기존 직전 결과 기반 문서 수정 흐름은 유지하면서, `선택한 md파일`처럼 선택을 명시한 요청만 현재 선택 파일을 우선하도록 조정했습니다.

## QA

- 현재 선택한 md 파일이 이전 xlsx 포커스보다 우선되는지 확인
- `영문으로 바꿔줘`가 치환이 아니라 영어 번역으로 분류되는지 확인
- xlsx 번역 요청 시 직접 저장 미지원 안내 확인
- 기존 파일/문서/이미지/압축/해상도/업스케일 작업 도우미 회귀 테스트
