# 파일정리함 (AssetFlow) 1.1.2

AI 파일 정보 질문과 파일/폴더 직접 작업 라우팅을 개선한 배포입니다.

## 주요 변경

- `zip 파일 몇 개야`, `압축파일 몇 개야`처럼 확장자/파일 종류를 붙인 질문을 실제 조건대로 계산합니다.
- `그중 제일 용량 큰 파일은?` 같은 후속 질문에서 직전 조건을 유지합니다.
- 개수가 맞지 않는다고 다시 확인하는 질문에는 실제 확인된 항목 목록을 함께 보여줍니다.
- `.crdownload`처럼 긴 확장자 파일명을 삭제/이동/복사 대상으로 인식합니다.
- 폴더 삭제, 폴더 이동, 폴더명 변경 같은 폴더 작업 명령 인식을 개선했습니다.
- 일반 질문/대화는 더 이상 `?` 규칙으로 막지 않습니다.
- AI 업스케일링과 로컬 AI 에이전트 설명 및 스크린샷을 공개 README에 반영했습니다.

## 포함 기능

- AI 업스케일링 모델 선택/찾기
- Real-ESRGAN 2x/4x 중심 업스케일링
- ONNX 모델 실행 지원
- 로컬 AI 에이전트 패널
- 파일 뷰어/이동, 이름/확장자 변경, 이미지 압축, 해상도 변경, 문서 변경
