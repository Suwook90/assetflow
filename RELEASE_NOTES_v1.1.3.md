# FilePilot AI 1.1.3

문서 변경 모드에서 선택 문서 번역/정리 결과가 잘못 저장되는 상황을 막는 안정화 배포입니다.

## 주요 변경

- `내용 영문으로 바꿔줘`, `영어로 번역해줘` 같은 요청을 영어 번역 모드로 우선 인식합니다.
- 로컬 AI가 `영문`, `English`처럼 지시어 한 단어만 반환하면 파일을 덮어쓰지 않고 다시 요청하도록 안내합니다.
- 문서 변환 프롬프트를 강화해 `output_text`에는 변환된 전체 문서 내용만 들어가도록 했습니다.
- 문서 변환 결과 저장 경로 추적을 정리했습니다.

## QA

- 문서 영어 번역 모드 인식
- 지시어 한 단어 반환 시 저장 차단
- 정상 영어 번역 결과 저장
- 기존 파일 정보/이동/삭제/이름변경/압축/해상도/업스케일 작업 도우미 회귀 테스트
