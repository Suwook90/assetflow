파일정리함 (AssetFlow) 0.1.3

한글 제품명과 오른쪽 대상 폴더 패널 UI를 정리한 업데이트입니다.

- 사용자 표시명을 `파일정리함`으로 변경
- 설치 화면, 시작 메뉴, 바탕화면 바로가기 이름을 한글 제품명 기준으로 정리
- 오른쪽 `옮길 폴더` 패널의 `현재 대상` 표시가 잘리지 않도록 2줄 라벨과 높이 조정
- 공개 저장소 README에 공개 다운로드 저장소와 비공개 소스 저장소 역할을 명확히 표시

주의: 현재 설치 파일은 코드 서명 인증서로 서명되지 않아 Windows SmartScreen 경고가 표시될 수 있습니다.
