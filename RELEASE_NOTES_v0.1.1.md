AssetFlow 0.1.1

문서 변경 모드의 개별 편집과 삭제 흐름을 보강한 업데이트입니다.

- 문서 변경 모드에서 선택한 TXT/JSON/MD 문서를 직접 편집하고 저장 가능
- `.bak 백업` 옵션을 켠 상태에서 개별 저장 시 원본 백업 생성
- 문서 목록에 실제 파일 삭제 버튼 추가
- 문서 목록 우클릭 메뉴에 `삭제` 추가
- 기존 `선택 비우기`는 목록에서만 제거하고, `삭제`는 파일을 휴지통 폴더로 이동하도록 구분
- 내부 안정성 점검 및 설치 파일 버전 갱신

주의: 현재 설치 파일은 코드 서명 인증서로 서명되지 않아 Windows SmartScreen 경고가 표시될 수 있습니다.
