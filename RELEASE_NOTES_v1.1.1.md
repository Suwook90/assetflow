# 파일정리함 (AssetFlow) 1.1.1

AI 업스케일링과 로컬 AI 에이전트 흐름을 포함한 1.1 계열 리뉴얼 릴리스입니다.

## 주요 변경

- AI 업스케일링 `선택 사용` 탭에서 설치된 모델의 before/after 미리보기를 오른쪽에 고정 표시합니다.
- `모델 찾기` 탭에서 현재 앱이 실행 가능한 모델만 검색하고, 선택한 실행 모델 파일 하나만 설치/사용합니다.
- ncnn Real-ESRGAN 모델과 유효한 ONNX 모델을 지원하고, `.pth`/`.safetensors`는 설치 대상에서 제외합니다.
- 1K/2K/4K 자동 제한, Sharp 후처리, 출력 형식/저장 폴더 설정을 지원합니다.
- 해상도 변경 모드에서 작은 원본을 단순 리사이즈할지 AI 업스케일 후 목표 해상도로 맞출지 선택할 수 있습니다.
- 로컬 AI 에이전트 패널을 각 작업 모드에 맞춰 사용할 수 있도록 정리했습니다.
- 표, 탭, 미리보기 카드, 위험 버튼, 툴 페이지 배경 등 UI/UX 스타일을 리뉴얼했습니다.
- README 스크린샷을 최신 화면으로 다시 생성하고 업스케일 모델 찾기 화면을 추가했습니다.

## 설치

- Windows 설치 파일: `AssetFlowSetup.exe`
- 다운로드: https://github.com/Suwook90/assetflow/releases/download/v1.1.1/AssetFlowSetup.exe

## 주의

현재 설치 파일은 코드 서명 인증서로 서명되지 않아 Windows SmartScreen 경고가 표시될 수 있습니다.
