AssetFlow 0.1.2

이미지가 많은 폴더에서 빠르게 이동할 때의 안정성을 보강한 버그픽스입니다.

- 파일 뷰어/이동 모드에서 화살표 키를 빠르게 누를 때 이미지 미리보기 작업이 과도하게 겹치지 않도록 개선
- 연속 선택 이동 중에는 최신 선택 항목만 일정 간격으로 미리보기 갱신
- 이미지 미리보기 로딩 실패 시 프로그램 종료 대신 화면에 오류 표시
- 창 크기 조정 중 이미지 미리보기도 과도하게 반복 로딩되지 않도록 조정

주의: 현재 설치 파일은 코드 서명 인증서로 서명되지 않아 Windows SmartScreen 경고가 표시될 수 있습니다.
